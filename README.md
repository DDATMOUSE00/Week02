# 회사에 늦지 않으려는 것 뿐인데 뭐시기뭐시기.. 

지각한 고릴라가 회사에 늦지 않기 위해 도시를 질주하는 2D 액션 게임입니다.  
플레이어는 차지 점프와 슬램을 활용해 빠르게 전진하고, 시민 무리와 수집 요소를 이용해 더 시원한 이동과 액션의 흐름을 만들어갑니다.

---

## 프로젝트 소개

이 프로젝트는 단순한 이동을 넘어,  
"달린다 → 날아간다 → 내려찍는다"의 반복이 계속해서 쾌감을 만들도록 설계한 액션 게임입니다.

고릴라의 묵직한 움직임, 포탄처럼 날아가는 추진력, 그리고 바닥을 강하게 찍어내리는 슬램을 중심으로  
짧은 시간 안에도 강한 인상을 남기는 플레이 경험을 목표로 했습니다.

또한 맵 위의 수집 요소인 붕어빵을 통해 플레이 동선에 변화를 주고,  
단순히 앞으로만 달리는 것이 아니라 순간순간의 판단과 욕심이 생기도록 구성했습니다.

---

## 핵심 재미

### 1. 차지 점프 기반의 시원한 이동
버튼을 눌러 힘을 모으고,  
방향성과 추진력이 실린 점프로 한 번에 멀리 도약할 수 있습니다.

### 2. 공중에서 이어지는 슬램 액션
점프 이후 공중에서 빠르게 내려찍는 슬램을 사용할 수 있습니다.  
슬램은 단순한 공격이 아니라, 이동의 리듬을 완성하는 핵심 액션입니다.

### 3. 밀도 있는 반복 플레이
달리기, 점프, 슬램의 반복만으로도 손맛이 살아나도록 구성했습니다.  
여기에 시민 무리, 붕어빵 수집, 지형과 속도감이 더해지며 플레이가 단조롭지 않게 이어집니다.

### 4. 수집 요소를 통한 동선 변화
맵 곳곳의 붕어빵은 플레이어가 자연스럽게 경로를 틀게 만드는 장치입니다.  
특정 개수 이상을 모으면 추가적인 보상을 얻을 수 있도록 설계했습니다.

---

## 게임 플레이 흐름

1. 앞으로 질주한다  
2. 차지 점프로 크게 도약한다  
3. 공중에서 슬램으로 빠르게 내려찍는다  
4. 속도를 유지한 채 다시 앞으로 이어간다  
5. 붕어빵과 시민 무리를 활용해 더 큰 리듬을 만든다

---

## 조작 방식

패드도 지원합니다.

| 입력 | 설명 |
|---|---|
| 이동 | 좌우 이동 |
| 점프 홀드 | 차지 점프 |
| 공중 입력 | 슬램 |
| 재시작 | R |

## 주요 시스템

- 차지량에 따라 점프의 추진력이 달라지는 이동 시스템
- 공중 상태에서 자연스럽게 이어지는 슬램 액션
- 플레이 흐름을 끊지 않도록 조정한 입력 보정
- 속도감과 무게감을 살리기 위한 카메라 연출
- 시민 AI의 도주 반응
- 붕어빵 수집 시스템
- 테스트 및 디버깅을 고려한 매니저 기반 구조

## 개발 목표

이 프로젝트에서 가장 중요하게 본 것은 단순히 "이동이 된다"가 아니라,  
플레이어가 입력했을 때 몸이 튕겨나가고, 떨어지고, 다시 이어지는 과정이 손에 잘 붙는가였습니다.

그래서 다음 요소를 중점적으로 다뤘습니다.

- 차지 점프의 추진력 조정
- 슬램의 타이밍과 후속 연결감
- 플레이 리듬을 해치지 않는 입력 유예
- 액션이 더 강하게 느껴지도록 만드는 카메라 및 이펙트 연출
- 수집 요소를 통한 순간적인 선택 유도

---

## 트러블슈팅 포인트

개발 과정에서는 아래와 같은 문제를 중점적으로 다뤘습니다.

- 점프와 착지 사이 입력 타이밍이 빡빡해 체감 조작감이 떨어지는 문제
- 공중 상태와 슬램 가능 상태의 분리 및 예외 처리
- 벽이나 지형에 닿았을 때 캐릭터가 부자연스럽게 붙는 문제
- 월드 이펙트가 보정 전 좌표에서 재생되는 문제
- 입력 장치에 따라 UI 힌트가 자연스럽게 교체되어야 하는 문제

이러한 문제들을 해결하면서,  
기능을 늘리는 것보다 플레이 감각을 정제하는 방향으로 시스템을 다듬었습니다.
